Los Ejemplos de Grid

Example 1

1. To define a Grid use display:grid or display:inline-grid on the parent element. You can then create a grid using the grid-template-columns and grid-template-rows properties. I am using the grid-gap property to create a gap between my columns and rows of 10px. This property is a shorthand for grid-column-gap and grid-row-gap so you can set these values individually. All direct children of the parent now become grid items and the auto-placement algorithm lays them out, one in each grid cell. Creating extra rows as needed.

1. Para definir una cuadrícula, utiliza display:grid o display:inline-grid en el elemento padre. Luego, puedes crear una cuadrícula usando las propiedades grid-template-columns y grid-template-rows.Estoy utilizando la propiedad grid-gap para crear un espacio entre mis columnas y filas de 10px. Esta propiedad es una forma abreviada de grid-column-gap y grid-row-gap, por lo que puedes establecer estos valores individualmente.Todos los hijos directos del padre ahora se convierten en elementos de la cuadrícula y el algoritmo de colocación automática los distribuye, uno en cada celda de la cuadrícula, creando filas adicionales según sea necesario.

Example 10

10. Explicit and Implicit Grid When we use grid-template-columns and grid-template-rows we create an Explicit Grid. However if we try and place an item outside of that grid the browser will create an Implicit Grid line or lines in order to hold that item. In the code below I have put e between grid column lines 4 and 5, these are not described with grid-template-rows, so an implicit grid line 5 is created. By default the implicit grid tracks created by the implicit lines will be auto sized. However, you can size the tracks with the grid-auto-columns and grid-auto-rows properties. I have sized my auto tracks as 100px to match the rest of the column tracks in my grid.

10. Cuadrícula Explícita e Implícita Cuando utilizamos grid-template-columns y grid-template-rows, creamos una cuadrícula explícita. Sin embargo, si intentamos colocar un ítem fuera de esa cuadrícula, el navegador creará una línea de cuadrícula implícita o líneas para mantener ese ítem. En el código a continuación, he puesto 'e' entre las líneas de columna 4 y 5, las cuales no están descritas con grid-template-rows, así que se crea una línea de cuadrícula implícita 5. Por defecto, las pistas de cuadrícula implícitas creadas por las líneas implícitas tendrán un tamaño automático. Sin embargo, puedes dimensionar las pistas con las propiedades grid-auto-columns y grid-auto-rows. He dimensionado mis pistas automáticas a 100px para que coincidan con el resto de las pistas de columna en mi cuadrícula.

11. Defining Grid Areas We can create named areas on the grid to put content into. To do this we first assign elements in our markup to a Grid Area, using the grid-area property. We can then use the grid-template-areas property to describe where on the Grid these elements should sit. Repeating the name of an area indicates that the area spans that grid track. Using a . or a sequence like .... indicates an empty track. Such as the very first cell on this grid.

11. Definiendo Áreas de la CuadrículaPodemos crear áreas nombradas en la cuadrícula para colocar contenido. Para hacer esto, primero asignamos elementos en nuestro marcado a un Área de la Cuadrícula, utilizando la propiedad grid-area.Luego podemos usar la propiedad grid-template-areas para describir dónde en la cuadrícula deben estar estos elementos.Repetir el nombre de un área indica que el área abarca esa pista de cuadrícula. Usar un punto (.) o una secuencia como .... indica una pista vacía. Tal como la primera celda de esta cuadrícula.

12. No clearing required If I add a footer to my markup and define a grid-area and place it on the grid using grid-template-areas you can see that it sits below the content column. There is no clearing required as this footer has a track on the grid, it can't hop up into another track. This example also shows the issue described in example 5, by default the background on the sidebar should stretch to full height, in a similar way to the defaults for Flexbox.

12. No se requiere limpiezaSi agrego un pie de página a mi marcado y defino un área de cuadrícula y la coloco en la cuadrícula utilizando grid-template-areas, puedes ver que se sitúa debajo de la columna de contenido. No se requiere limpieza ya que este pie de página tiene una pista en la cuadrícula, no puede saltar a otra pista.Este ejemplo también muestra el problema descrito en el ejemplo 5, por defecto el fondo en la barra lateral debería extenderse hasta la altura completa, de manera similar a los valores predeterminados para Flexbox.
